# Sandcastle Blueprints

SVG blueprint designs for competitive/recreational sandcastle building. Covers crew equipment and job roles for 2-crew operations.

## Crew Equipment (per crew)

| Item | Description |
|------|-------------|
| Shovel | For scooping and loading sand into the cylinder |
| Water Bucket | 5-gallon bucket for hauling water from the source |
| Cylinder | Bottomless 5-gallon bucket used as a form for packing sand/water mixture |

## Job Roles (rotating)

These roles rotate frequently among crew members:

| Role | Responsibility |
|------|---------------|
| Water Hauler | Fetches water and pours it into the cylinder |
| Sand Loader | Shovels sand into the cylinder |
| Stomper | Compacts the water/sand mixture inside the cylinder by stomping |
| Brusher | Uses paintbrushes to remove loose sand from the formed block |
| Carver | Sculpts and carves detail from the compacted sand block |
